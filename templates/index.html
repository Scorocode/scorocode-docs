Документация облачного сервиса scorocode

Добро пожаловать на страницу документации облачного scorocode!

В документации представлены:

  • описание работы с личным кабинетом, приложениями, данными
  • документация по HTTP API для реализации работы с облаком на любой платформе
  • документация по SDK: WEB сервер/клиент (JavaScript SDK), iOS/Mac OS (Swift SDK). Android (Java SDK) на подходе.

Исходные коды всех SDK, а так же самого документации открыты и размещены на GitHub. Все замечания, изменения и предложения по каждому репозиторию категорически приветствуются.

Для перехода в нужный раздел документации выберите его в левом меню.
