    .Init(opt) - Инициализация SDK¶

    Параметр Тип Описание
    opt Object Ключи для инициализации: идентификатор приложения, клиентский ключ вашей платформы и необходимые вам ключи доступа.

    Свойства

    Имя Тип Описание
    ApplicationID String Идентификатор приложения
    JavaScriptKey String Клиентский ключ для платформы JavaScript
    AndroidKey String Клиентский ключ для платформы Android
    iOsKey String Клиентский ключ для платформы iOs
    WinPhoneKey String Клиентский ключ для платформы WindowsPhone
    FileKey String Ключ авторизации для доступа к файлам
    ScriptKey String Ключ авторизации для запуска серверных скриптов
    MessageKey String Ключ авторизации для отправки сообщений
    WebSocketKey String Ключ авторизации для открытия WebSocket-соединения
    MasterKey String Мастер-ключ приложения

    Пример 

    Scorocode.Init({
    ApplicationID: "a3d04e75e157b2f7ae20c2fce02f63d6",
    JavaScriptKey: "ad6a8fe72ef7dfb9c46958aacb15196a",
    FileKey: "2aceceec7d2e96b1487ec399f5afa101",
    MessageKey: "e215ed465646775b42d65cca2d2f5db9"
});
